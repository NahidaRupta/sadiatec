# Brand Assets

Place the following files in this directory before deployment:

| File | Size | Notes |
|------|------|-------|
| `logo.svg` | max 400×120px | Must include accessible `<title>` text inside the SVG |
| `favicon.ico` | 32×32px | ICO format with 16px and 32px variants |
| `og-image.jpg` | 1200×630px | Used for Open Graph previews |
| `brandmark.svg` | optional | Square version of logo for small contexts |

These files are gitignored and must be added per-client deployment.
